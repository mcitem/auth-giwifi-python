# 使用

Windows下：

```
pip install pycryptodome
pip install requests
pip install bs4
```

```
python master.py username password ip(可选)
```

打包版本
```
.\master.exe username password ip(可选)
```

Linux下：
```
pip install pycryptodome
pip install requests
pip install bs4
```

```
python master.py username password ip(可选)
```

打包版本
```
chmod 777 master
./master username password ip(可选)
```
